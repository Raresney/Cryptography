# 🔐 Cryptography

A collection of cryptography exercises, implementations and experiments.

This repository contains implementations of basic cryptographic concepts and algorithms, developed for learning and experimentation purposes.

The goal of this project is to understand the **mathematical foundations of cryptography** and how encryption algorithms work in practice.

---

## 📚 Topics Covered

This repository may include implementations or experiments related to:

- Classical cryptography
- Modular arithmetic
- Prime number testing
- Public key cryptography
- Cryptographic algorithms
- Encryption / Decryption techniques

---

## 🧠 Learning Objectives

The purpose of this repository is to:

- Understand the **mathematical principles behind cryptography**
- Implement cryptographic algorithms from scratch
- Explore number theory used in encryption
- Practice secure algorithm design

## 🛠 Technologies

- Python
- Mathematical algorithms
- Number theory
- Cryptographic primitives
